## Solutions to *A Classical Introduction to Modern Number Theory* by Ireland and Rosen

This is a partial solution guide to the exercises from the book *A Classical Introduction to Modern Number Theory* (2nd ed.) by Ireland and Rosen. Since this paper was done by myself solely, there's no guarantee that all the proofs are 100% correct.
